Inserindo JS na sua página

Você pode inserir JS em sua página de duas formas:

  1. Interno: dentro de uma tag <script>
  2. Externo: em um arquivo separado
Onde é colocado o script
  • No cabeçalho do documento
    • É lido antes do documento ser rederizado, e o código é executado imediatamente.
  • No body do documento HTML
    • É lido e executado na ordem em que aparece no documento.
  • imediatamente antes do body
    • Nesse caso, o código é executado depois que o documento foi lido e renderizado.

É possível inserir vários scripts em diferentes regiões de uma mesma página.

Os scripts do JS podem ser executados independentes do HTML utilizando o Node.Js instalado no Sistema Operacional.

Valores e Tipos

Tipagem dinâmica: os tipos são assiciados a valores, e não a variáveis. Os tipos associados são:

  • Number (interios ou pontos flutuantes)
  • String (pode-se usar aspas semples ou duplas)
  • Boolean
  • Object
  • Undefined (para variaveis não inicializadas)
  • bigint (ES2020)

O tipo de uma variável pode ser checada por meio do typeof

Tipo Number

O tipo number não tem separação, ele pode ser inteiro ou real, dependerar do valor atribuido a variável. Já o bigint é apenas para inteiros

Valores Numéricos Especiais

  • Infinity 3/0
  • NaN Not a Number
  • Number.MAX_VALUE Maior número positivo (float)
  • Number.MIN_VALUE Menor número positivo maior que zero

Objeto Math

É um objeto pré-definido que tem propriedades e métodos para constantes e funções matemáticas

A linguagem JavaScript possui inúmeros outros exemplos de objetos pré-definidos tais como Date, String, Array, Number, JSON, etc.

Tipo BigInt

  • Como vimos, o maior tipo Number inteiro aceito é 2^53-1 – 9007199254740991, cerca de 9 quatrilhões
  • O ES2020 criou um novo tipo para aumentar a capacidade dos inteiros
  • Para criar um BigInt, basta adicionar n ao final do número inteiro, ou então usar o contrutor BigInt()
  • Segunda a especificação, o tipo BigInt é um tipo primitivo capaz de representar valores inteiros de precisão arbitrária, mas qualquer representação sofre limitações de hardware

Tipo String

  • Manipuladores de pedaçoes de textos.
  • Imutáveis
  • Após a variável ser declarada, passa a ter um conjunto de métodos e propriedades.(imutáveis)

Templates String

  • Templates String permitem a criação de strings com valores e expressões embutidas.

Valores Booleanos

  • Assume verdadeiro ou falso.
  • Operadores também podem produzir valores booleanos.
    • Operadores lógicos binários: && (and) e || (or)
    • Operadores prefixos lógicos: ! (Not)
    • Operadores de comparação:
      • Operadores de equidade: ===, !==, == ,!=
        • A diferença entre == e === é que == compara apenas os valores das variáveis, enquanto === compara os valores e os tipos de valores
      • Operadores de ordem: >, >=, <, <=
  • Alguns valores são considerados falsos em JS.
    • 0, -0, NaN
    • "" (string vazia)
    • null, Undefined
    • false

Avaliação de curso-circuito

As expressões são analizadas da esquerda para direita, então se o primeiro valor lógico for suficiente para determinar o valor da expressão, o segundo elemento é ignorado.

Uma aplicação do uso de avaliações em curto circuito é condicionar a execução de uma função ao retorno de outra função: primeraFuncao() && segundaFuncao()x

Condicionais

  • O comando if possui uma cláusula opcional else que é executada dependendo de uma condição booleana
  • Opera com condicinais ternários (test? expression1 : expression2)
  • O switch também é suportado, valor da variável define qual case será executado.
  • Os loops da linguagem JS são os mesmos das liguagens C e Java:
    • for(val i = 0; i < valor; i++)
    • while
    • do while
