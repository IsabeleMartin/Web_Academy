 A semântica do HTML é a forma   como se estrutura o conteúdo de uma página com base no significado de cada elemnento, e não apenas na aparência

O que é semântica HTML?

Semântica HTML significa usar as tags (elementos) HTML de forma que o nome delas indique o seu
e propósito no conteúdo da página.

Em outras palavras, você está dizendo ao navegador (e também a leitores de tela, mecanismos de busca
como o Google, etc.) “o que é cada parte do conteúdo”, e não só como ela deve parecer.

Por que Semântica é importante?

  1. Acessibilidade: Pessoas com deficiência conseguem navegar melhor pela página.
  2. SEO(Otimização para moteres de busca): Motores de busca entendem melhor sua página, melhorando o ranqueamento.
  3. Organização e Manutenção: O código fica mais fácil de entender e manter.
  4. Compatibilidade: Navegadores e dispositivos diferentes interpretam o conteúdo de forma mais consistente.
