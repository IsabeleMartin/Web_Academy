Layout Flexbox

O flexbox é um método display que permique que organizamos itens de uma página dentro de um container.

Algumas propriedades CSS do flexbox estão relacionadas com o container, enquanto outras com os itens

Propriedade do Flex Container

  • display: define um flex container, habilita o contexto flex para todos os filhos (flex itens internos)
.container {display: flex | inline-flex;}
  • flex-direction: estabelece o main-axis, definindo a direção dos itens dentro de um flex container.
.container { flex-direction: row | row-reverse | column | column-reverse; }
  • row: dispoe os elementos em uma linha da esquerda para direita.
  • row-reverse: dispoe os elementos em uma lista da direita para esquerda.
  • column: dispoe os elementos em uma coluna de cima para baixo.
  • column-reverse: dispoe os elementos em uma coluna de baixo para cima.
