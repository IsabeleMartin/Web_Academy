    • Tamanho dos Elementos

    As propriedades usadas para definir o tamanho de um elemento são:

    • width
    • height
    • max-width
    • max-height
    • min-width
    • min-height

    O valor padrão para todas elas é o auto, onde o browser calcula o tamanho do elemento de acordo com o conteúdo que ele possui. Para definir um tamanho fixo, você pode usar pixels (px), porcentagem (%) ou outras unidades de medida.

    min- e max-

    Utilizado para limitar o quanto o browser pode aumentar ou diminuir o tamanho do elemento.

    Lidando com Transbordamento

    Quando um elemento é diminuido, o seu conteudo pode transbordar para fora, por padrão o browser permite que ele transborde, mas existem meios de tratamento para isso utilizando a propriedade overflow.

    • overflow-x
    • overflow-y
    • overflow

    Possíveis valores para a propriedade overflow:

    Valor Descrição
    visible O conteúdo transbordado é visível (valor padrão).
    hidden O conteúdo transbordado é cortado e não é visível.
    scroll Adiciona barras de rolagem havendo ou não transbordamento.
    auto Adiciona barras de rolagem apenas se necessário.

    Alinhamento de Textos

    A propriedade text-align é usada para configurar o alinhamento do conteúdo textual. Seus possíveis valores são: left, right, center e justify.

    Além da propriedade text-align, existe uma série de outras propridades de textos: text-indent, text-transform, text-decoration, word-spacing e letter-spacing.

    Posicionamento de Conteúdo

    A propriedade position define o método com que um elemento é posicionado na página. Possíveis valores:

    Valor Descrição Obeservação
    static O elemento é definido conforme o fluxo do documento. Valor padrão. Top, Bottom, Left e Right não tem efeito
    relative O elemento ocupa seu espaço original, mas pode mover com top, left, etc Serve de referência para elementos absolute filhos.
    absolute O elemento é retirado do fluxo do documento e é posicionado em relação ao ancestral mais próximo. O ancestral deve ser diferente de static
    fixed O elemento fica fixo na tela do navegador, não se move com scroll. ideal para Cabeçalhos fixos, botão de voltar ao topo.

    Posicionamento Absoluto

    O posicinamento absoluto permite ao programador maior controle sobre o local que o elemento ficará na página. O padrão é que o elemento siga o fluxo do documento, mas quando é especificado a posição absoluta, isso é ignorado. Os elementos são posicinados de acordo com a distância do topo, esquerda, direita ou do fundo da página.

    Vamos colocar o gato tutor para brincar na caixa de areia, aperte o botão fixed e role a página até ele entrar na caixa.

    Opa, o gato tutor está posicionado atrás da caixa de areia. Isso ocorre pois a disposição das imagens segue a ordem dos últimos que foram insridas na página. Para resolver isso, é utilizado a propriedade z-index que permite criar elementos sobrepostos.

    Vejamos o exemplo abaixo: das duas imagens, a primeira é a caixa de areia e a segunda é o gato tutor. O gato auxiliar está com o z-index 2 e a caixa de areia com o z-index 1, ou seja, o gato tutor está atrás da caixa de areia.
