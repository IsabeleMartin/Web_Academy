Objetos

Em JavaScript, um objeto é uma coleção dinâmica de pares chave-valor, onde os valores podem ser de qualquer tipo de dado.

Exemplo de Objeto


    let ufam = {
    nome: "Universidade Federal do Amazonas",
    fundacao: 1909,
    ativa: true
    };

    document.writeln(ufam.nome);
    document.writeln(ufam.fundacao); 
    document.writeln(ufam.ativa);

    document.writeln("<br>");

    document.writeln(ufam["nome"]);    
    document.writeln(ufam["fundacao"]);
    document.writeln(ufam["ativa"]);

As propriedades do objeto podem ser acessadas de duas formas: por meio de ponto (ufam.nome) ou por colchetes (ufam["nome"]).

Também é possível utilizar uma variável com o nome da propriedade, como em ufam[variavel].

O uso de colchetes é obrigatório quando o nome da propriedade contém espaços ou começa com números.

Propriedades podem ser adicionadas ou removidas dinamicamente:

  • Remover: delete ufam.fundacao;
  • Adicionar: ufam.sigla = "UFAM";

Também é possível definir funções dentro de objetos:

Exemplo de Função em Objeto


    let ufam = {
    nome: "Universidade Federal do Amazonas",
    fundacao: 1909,
    ativa: true,
    imprime: function () {
        document.writeln(this.nome);
    }
    };

    ufam.imprime();

A palavra-chave this refere-se ao próprio objeto, permitindo acessar suas propriedades e métodos.

É possível percorrer cada propriedade de um objeto com um laço for..in:

Exemplo de for..in


    for (let prop in ufam) {
    document.writeln(prop + ": " + ufam[prop] + "<br>");
    }

Quando uma função é percorrida dessa forma, seu conteúdo é convertido em string e exibido.

O operador in pode ser usado para verificar se uma propriedade existe em um objeto:

Exemplo de operador in


    if ("nome" in ufam) {
    document.writeln("A propriedade nome existe");
    } else {
    document.writeln("A propriedade nome não existe");
    }

Arrays

Os Arrays diferentemente dos objetos citados acima, não possuem valores referenciados por propriedades/chaves, ele são referenciados por posição numérica.

    
let arr = [
    "Universidade Federal do Amazonas",
    "1909",
    true
];
document.writeln(arr[0] + "<br>");
document.writeln(arr[1] + "<br>");
document.writeln(arr[2] + "<br>");
document.writeln(arr.length + "<br>");
document.writeln(typeof arr);

Formas de criar um array


let arr = [
    "Universidade Federal do Amazonas",
    1909,
    true
];
let arr = [];
arr[0] = "Universidade Federal do Amazonas",
arr[1] = 1909,
arr[2] = true

let arr = new Array();
arr[0] = "Universidade Federal do Amazonas",
arr[1] = 1909,
arr[2] = true

    true

arr.forEach(function (valor) {
    document.writeln(valor + "
");

Funções

Assim como o array, as funções também são objetos. Existem três métodos de definir uma função

Exemplo de função


/* Declaração de Função */
function imprimir (str){
    console.log(str);
}

/*Expressão de função*/
const imprimir = function (str){
    console.log(str);
};

/*vetor de função*/
const imprimir = (str) => {
    console.log(str);
};

Funções de primeira classe

  • Ações permitidas por funções de primeira classe:
    • Atribuir uma função a uma variável
    
const hello = function() {
return "Hello World";
}
    • Retornar uma função como valor de outra função
    
const hello = function() {
    return function() {
        return "Hello World";
    };
}
    
console.log(hello);
console.log(hello());
console.log(hello()());
    • Passar uma função como parâmetro para outra função
    
const produto = {
    tipo: 'computador',
    preco: 2750.0
}
    
const impostoPessoaFisica = function (preco) { 
    return preco * 0.15;
}
    
const impostoPessoaJuridica = function (preco) { 
    return preco * 0.12;
}
    
const calculaPreco = function (produto, formulaImposto) {
    return produto.preco + formulaImposto(produto.preco)
} 
    
calculaPreco(produto,impostoPessoaFisica); // 3162.5
calculaPreco(produto,impostoPessoaJuridica); // 3080

Escopo em JavaScript

1. Escopo Global e Local

Exemplo de variáveis definidas dentro e fora de funções

Variáveis definidas fora de funções são globais. Dentro de funções, são locais.

let globalVar = "sou global";
      
      function teste() {
        let localVar = "sou local";
        console.log(globalVar); // OK
        console.log(localVar);  // OK
      }
      
      teste();
      console.log(localVar); // ❌ Erro

2. var, let e const

Comparação entre escopos de var, let e const

Diferenças de escopo e comportamento.

if (true) {
        var x = 10;
        let y = 20;
        const z = 30;
      }
      
      console.log(x); // ✅
      console.log(y); // ❌
      console.log(z); // ❌

3. var em loops

Escopo de var e let dentro de estruturas de repetição

for (var i = 0; i < 3; i++) {
        // faz algo
      }
      console.log(i); // ✅ 3
      
      for (let j = 0; j < 3; j++) {
        // faz algo
      }
      console.log(j); // ❌ Erro

4. Acesso fora do escopo

Exemplo de erro ao acessar variável fora do escopo

function exemplo() {
        let interna = 5;
      }
      console.log(interna); // ❌ ReferenceError

5. Funções Imediatas (IIFE)

Execução imediata e isolamento de escopo

(function () {
        let segredo = "oculto";
        console.log("Executado imediatamente");
      })();
      
      console.log(segredo); // ❌ Erro

6. Cadeia de Escopos

Como o JavaScript procura variáveis nos escopos pais

let valor = "global";
      
      function pai() {
        let valor = "pai";
        function filho() {
          console.log(valor); // "pai"
        }
        filho();
      }
      pai();

7. Closures

Funções internas que lembram o escopo onde foram criadas

function contador() {
        let count = 0;
        return function () {
          count++;
          console.log(count);
        };
      }
      
      let incrementar = contador();
      incrementar(); // 1
      incrementar(); // 2
      incrementar(); // 3
